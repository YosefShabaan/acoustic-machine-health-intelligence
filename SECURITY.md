# Security

This repository is a research prototype, not a production security-certified system.

Do not include secrets, credentials, private maintenance documents, raw datasets, WAV files, model weights, generated indexes, or generated scientific artifacts in issues, pull requests, commits, or screenshots.

If you find a security concern, report it privately through the repository owner's GitHub security/private contact path rather than opening a public issue with sensitive details.

Security review does not imply the system is ready for production deployment. Production use would require a separate security, privacy, data-governance, and operational-readiness review.
